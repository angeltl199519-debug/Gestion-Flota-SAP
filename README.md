# 🚛 SISTEMA DE GESTIÓN DE MANTENIMIENTO DE FLOTA - SAP 2.0

## 📦 Contenido del Sistema

Este es un sistema completo y profesional para gestionar el mantenimiento de 20-40 unidades de transporte de mercancías e hidrocarburos.

### Archivos Incluidos:

1. **Gestion_Flota_SAP_Completo.py** - Script generador del Excel
2. **Generar_Reportes_PDF.py** - Generador de reportes en PDF
3. **Macros_VBA_Excel.bas** - Macros para Excel
4. **README.md** - Esta guía

## ⚡ Instalación Rápida

```bash
# 1. Instalar dependencias
pip install openpyxl reportlab sqlite3

# 2. Generar el sistema
python Gestion_Flota_SAP_Completo.py

# 3. Generar reportes
python Generar_Reportes_PDF.py
```
